# Online Tender System

A simple tender management system built with:
- Angular (Frontend)
- Spring Boot (Backend)
- MySQL (Database)

## Features
✔ User authentication  
✔ Product management  
✔ Bidding system  

## Setup
1. Backend: `mvn spring-boot:run`
2. Frontend: `ng serve`
